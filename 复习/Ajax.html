//xhr的就绪状态有：xhr.readyState //xhr.readyState === 0 创建了，没open //xhr.readyState === 1 open了，没send //xhr.readyState === 2 send了，数据还没返回 //xhr.readyState === 3 send了，返回部分数据 //xhr.readyState === 4 send了，数据全部返回 //响应的状态码：xhr.status // 200 OK // 404 NOT FOUND // 302 REDIRECT // 5XX 服务器错误 //状态码意义： // 2xx 响应成功 // 3xx 请求被重定向 // 4xx 请求错误 // 5xx 服务器错误
